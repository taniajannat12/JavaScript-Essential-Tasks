JS Practice Kit - Interactive Logic Showcase
This project is an interactive one-page dashboard designed to demonstrate various JavaScript functionalities and logical implementations. It uses a clean user interface styled with Tailwind CSS v4.

🛠 Implemented Logic & Features
Dynamic Navigation (showTask): Implements a section-switching logic using classList.add/remove to toggle the visibility of different task cards without reloading the page.

Theme Customization (themeToggle): A state-checking logic using classList.contains and classList.replace to switch between light and dark modes dynamically.

Real-time Greetings: Captures user input and utilizes innerHTML to generate immediate personalized welcome messages.

Interactive Counter: Uses global variable management for incrementing/decrementing values, combined with conditional logic to change text colors based on positive or negative states.

RGB Random Color Generator: Employs Math.random() and Math.floor() to generate dynamic RGB values that update the document's background color in real-time.

Mathematical Wizardry: Performs simultaneous square and cube calculations by processing numeric input through arithmetic operators.

Content Filtering Logic: Uses a loop to scan element classList properties, allowing users to filter items (e.g., Fruits vs. Vegetables) by toggling their display style.

Email Pattern Validation: A basic validation logic using includes() to verify the presence of essential characters ("@" and ".com") with visual feedback via border-color changes.

Random Quote Engine: Picks a random index from a predefined array to render motivational text dynamically.

Live Data Injection: Utilizes DOM manipulation through insertRow() and querySelector to append user-submitted form data directly into a table.

💻 Technology Stack
Frontend: HTML5

Styling: Tailwind CSS v4 (via CDN)

Scripting: Vanilla JavaScript (ES6+)

How to Run the Project:

Clone or download the repository.

Open the index.html file in any modern web browser.

Use the sidebar to navigate through the different tasks and test the logic.
